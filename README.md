# ImageCompressToolAngular

- Compress any image with this app. 
- This app is live [at](https://rd003.github.io/image-compress-tool-angular/)

## Run it in development environment

```bash
# clone repo
git clone https://github.com/rd003/image-compress-tool-angular.git

# visit directory
cd image-compress-tool-angular

# install packages
nmp i

# run the app
ng serve --open
```

## Tech used

- Angular 21
- browser-image-compression (library used for compression)
